# 🎬 PelixStream

![PelixStream Banner](https://images.unsplash.com/photo-1536440136628-849c177e76a1?q=80&w=1920&auto=format&fit=crop)

¡Bienvenido a **PelixStream**! Tu plataforma definitiva para ver y descargar películas, series y anime en español latino y máxima calidad HD.

## 🚀 Características Principales

- **Diseño Premium**: Interfaz de usuario moderna, rápida y oscura inspirada en las mejores plataformas de streaming (Netflix, HBO Max).
- **SEO Automático**: Generación de etiquetas dinámicas (Open Graph, Twitter Cards) por cada película para máxima visibilidad en Google.
- **Buscador Inteligente**: Encuentra títulos al instante sin recargar la página.
- **Secciones Dinámicas**: Páginas dedicadas y generadas automáticamente para Películas, Series, Anime y Géneros.
- **Panel de Administración**: Gestión de contenido integrado con TMDB y Supabase. Añade películas en segundos.
- **Sitemap Dinámico**: Generación en tiempo real de `sitemap.xml` para motores de búsqueda.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: Next.js 14 (App Router), React, TailwindCSS, Framer Motion
- **Backend & Base de Datos**: Supabase (PostgreSQL)
- **APIs**: TMDB (The Movie Database) para metadatos e imágenes
- **Iconos**: Lucide React

## 📱 Instalación Local

1. Clona el repositorio:
```bash
git clone https://github.com/nicolaszierow1021-alt/streampel.git
```
2. Instala las dependencias:
```bash
npm install
```
3. Configura tus variables de entorno en un archivo `.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=tu_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_clave
```
4. Inicia el servidor de desarrollo:
```bash
npm run dev
```
