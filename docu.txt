Endpoints de Listado
Autenticación Requerida
Todos los endpoints de listado requieren autenticación mediante API Key. Debes incluir tu API Key en el header X-API-Key en cada solicitud.

Listar Películas
Obtiene un listado paginado de películas disponibles en el sistema, ordenadas por fecha de sincronización descendente.

Endpoint: GET https://vimeus.com/api/listing/movies

Query Parameters
page
opcional
integer
Número de página a recuperar. Por defecto es 1. Cada página contiene hasta 50 resultados.
Headers Requeridos
X-API-Key: tu_api_key_aqui

Notas Técnicas
•
Los resultados están ordenados por fecha de sincronización descendente (más recientes primero).
•
El límite de resultados por página es fijo en 50 elementos para mantener tiempos de respuesta óptimos.
•
Las URLs de imágenes (poster y backdrop) son rutas relativas de TMDB. Deben ser concatenadas con: https://image.tmdb.org/t/p/original
•
El uso de la API Key se registra en el sistema para fines de auditoría y estadísticas.
•
Las respuestas incluyen un objeto de paginación completo con información útil para implementar navegación en tu aplicación.
PHP SDK
Se requiere autenticación
Todas las solicitudes requieren una API Key a través de X-API-Key. No expongas tu API Key en el código del lado del cliente.

La siguiente Sdk Class encapsula las llamadas HTTP con encabezados consistentes, manejo de errores y valores predeterminados. Es independiente del framework y compatible con PHP moderno.

API Key vs View Key
API Key: Autenticación para consultar la API y obtener listados/embeds (servidor)
View Key: Ya incluida en los embed_url retornados. Valida acceso al iframe desde dominios permitidos
Instalación
Copia la clase en tu proyecto (e.g., src/VimeusSdk.php) e inclúyelo mediante autoload de Composer o require manual.

PHP SDK
<?php
declare(strict_types=1);

/**
 * Lightweight API SDK for Vimeus.
 *
 * - Uses cURL with timeouts and JSON handling.
 * - Sends X-API-Key header on each request.
 * - Returns embed_url with view_key already included.
 * - Throws RuntimeException on transport or HTTP errors.
 *
 * Usage:
 *   $sdk = new VimeusSdk(apiKey: 'your_api_key', baseUrl: 'https://vimeus.com');
 *   $movies = $sdk->listMovies(page: 1);
 */
final class VimeusSdk
{
    /** @var string */
    private $apiKey;

    /** @var string */
    private $baseUrl;

    /** @var int Timeout in seconds */
    private $timeout;

    /**
     * @param string $apiKey  Your API key (X-API-Key header).
     * @param string $baseUrl API base URL (e.g., https://vimeus.com)
     * @param int    $timeout Request timeout (seconds).
     */
    public function __construct(string $apiKey, string $baseUrl = 'https://vimeus.com', int $timeout = 15)
    {
        if ($apiKey === '') {
            throw new \InvalidArgumentException('API key must not be empty.');
        }
        if ($baseUrl === '' || !preg_match('#^https?://#i', $baseUrl)) {
            throw new \InvalidArgumentException('Base URL must be an absolute HTTP(S) URL.');
        }

        $this->apiKey = $apiKey;
        $this->baseUrl = rtrim($baseUrl, '/');
        $this->timeout = $timeout;
    }

    /**
     * Perform an HTTP request and return decoded JSON as array.
     *
     * @param string               $method   HTTP method (GET, POST, etc.)
     * @param string               $path     Absolute or relative path under base URL.
     * @param array<string, mixed> $query    Query parameters.
     * @param array<string, mixed>|null $json Optional JSON payload for non-GET.
     *
     * @return array<string, mixed>
     */
    private function request(string $method, string $path, array $query = [], ?array $json = null): array
    {
        $method = strtoupper($method);
        $url = $this->buildUrl($path, $query);

        $ch = curl_init($url);
        if ($ch === false) {
            throw new \RuntimeException('Failed to initialize cURL.');
        }

        // Common headers.
        $headers = [
            'Accept: application/json',
            'X-API-Key: ' . $this->apiKey,
            'User-Agent: VimeusSdk-PHP/1.0',
        ];

        // Configure method and optional body.
        if ($method !== 'GET') {
            curl_setopt($ch, CURLOPT_CUSTOMREQUEST, $method);
            if ($json !== null) {
                $body = json_encode($json, JSON_UNESCAPED_SLASHES | JSON_UNESCAPED_UNICODE);
                if ($body === false) {
                    curl_close($ch);
                    throw new \RuntimeException('Failed to encode JSON body.');
                }
                $headers[] = 'Content-Type: application/json';
                curl_setopt($ch, CURLOPT_POSTFIELDS, $body);
            }
        }

        // Transport options.
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
        curl_setopt($ch, CURLOPT_CONNECTTIMEOUT, $this->timeout);
        curl_setopt($ch, CURLOPT_TIMEOUT, $this->timeout);
        curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);

        $raw = curl_exec($ch);
        $errno = curl_errno($ch);
        $error = curl_error($ch);
        $status = (int) curl_getinfo($ch, CURLINFO_HTTP_CODE);
        curl_close($ch);

        // Transport-level error.
        if ($errno !== 0) {
            throw new \RuntimeException(sprintf('Network error (%d): %s', $errno, $error));
        }

        // Decode JSON; empty body becomes empty array.
        $decoded = [];
        if (is_string($raw) && $raw !== '') {
            $decoded = json_decode($raw, true);
            if (!is_array($decoded)) {
                throw new \RuntimeException('Invalid JSON response.');
            }
        }

        // HTTP-level error handling with message propagation.
        if ($status < 200 || $status >= 300) {
            $msg = isset($decoded['message']) ? (string) $decoded['message'] : 'HTTP error';
            throw new \RuntimeException(sprintf('HTTP %d: %s', $status, $msg));
        }

        /** @var array<string, mixed> $decoded */
        return $decoded;
    }

    /**
     * Build absolute URL with query string.
     *
     * @param string               $path
     * @param array<string, mixed> $query
     */
    private function buildUrl(string $path, array $query): string
    {
        $path = '/' . ltrim($path, '/');
        $url = $this->baseUrl . $path;

        if (!empty($query)) {
            $filtered = array_filter($query, static function ($v) {
                return $v !== null;
            });
            $qs = http_build_query($filtered, '', '&', PHP_QUERY_RFC3986);
            if ($qs !== '') {
                $url .= '?' . $qs;
            }
        }
        return $url;
    }

    // ---- Listing endpoints (return embed_url with view_key included) ------

    /**
     * GET /api/listing/movies
     * Returns array with movies and their embed_url (view_key already included).
     *
     * @param int $page Page number (default 1).
     * @return array<string, mixed>
     */
    public function listMovies(int $page = 1): array
    {
        return $this->request('GET', '/api/listing/movies', ['page' => $page]);
    }

    /**
     * GET /api/listing/series
     * Returns array with series and their embed_url (view_key already included).
     *
     * @param int $page Page number (default 1).
     * @return array<string, mixed>
     */
    public function listSeries(int $page = 1): array
    {
        return $this->request('GET', '/api/listing/series', ['page' => $page]);
    }

    /**
     * GET /api/listing/animes
     * Returns array with animes and their embed_url (view_key already included).
     *
     * @param int $page Page number (default 1).
     * @return array<string, mixed>
     */
    public function listAnimes(int $page = 1): array
    {
        return $this->request('GET', '/api/listing/animes', ['page' => $page]);
    }

    /**
     * GET /api/listing/episodes
     * Returns array with episodes and their embed_url (view_key already included).
     *
     * @param int      $page    Page number (default 1).
     * @param int|null $tmdbId  Optional TMDB series ID filter.
     * @param int|null $season  Optional season filter (requires tmdbId).
     * @return array<string, mixed>
     */
    public function listEpisodes(int $page = 1, ?int $tmdbId = null, ?int $season = null): array
    {
        $query = ['page' => $page, 'tmdb_id' => $tmdbId, 'season' => $season];
        return $this->request('GET', '/api/listing/episodes', $query);
    }
}
Uso básico
<?php
require __DIR__ . '/vendor/autoload.php';

$sdk = new VimeusSdk(apiKey: 'your_api_key', baseUrl: 'https://vimeus.com');

try {
    // Obtener listado de películas (página 1)
    $movies = $sdk->listMovies(page: 1);
    
    // La respuesta incluye embed_url con view_key ya incluida:
    // {
    //   "data": [
    //     {
    //       "tmdb_id": 550,
    //       "title": "Fight Club",
    //       "embed_url": "/e/movie?tmdb=550&view_key=abc123xyz"
    //     }
    //   ]
    // }
    
    foreach ($movies['data'] as $movie) {
        $embedUrl = $movie['embed_url'];
        // Usar directamente en iframe
        echo '<iframe src="' . htmlspecialchars($embedUrl) . '" referrerpolicy="origin"></iframe>';
    }
    
} catch (RuntimeException $e) {
    error_log($e->getMessage());
    http_response_code(500);
    echo 'Request failed.';
}
Personalización de player
Si has configurado personalización en tu player (tema, logos, splash), concatena tus query params al embed_url:

<?php
// La API retorna: /e/movie?tmdb=550&view_key=abc123xyz
$embedUrl = $movie['embed_url'];

// Agregar personalización de player
$customParams = http_build_query([
    'theme' => 'vimeus',
    'logo_url' => 'https://tu-web.com/logo.png',
    'primary_color' => '10b981'
]);

$finalUrl = $embedUrl . '&' . $customParams;
// Resultado: /e/movie?tmdb=550&view_key=abc123xyz&theme=vimeus&logo_url=...

echo '<iframe src="' . htmlspecialchars($finalUrl) . '" referrerpolicy="origin"></iframe>';
Notas técnicas
•
API Key: Autentica tus llamadas al servidor. Úsala solo en backend, nunca en frontend.
•
View Key: Ya está incluida en el embed_url. Valida que el iframe se use solo en dominios permitidos.
•
Referrer Policy: Siempre usa referrerpolicy="origin" en tus iframes para validación correcta.
•
Los timeouts HTTP son ajustables mediante el constructor (conexión y totales).
•
Los errores lanzan RuntimeException con mensajes descriptivos.
Flujo completo
Genera tu API Key desde el dashboard
Consulta los endpoints de listing (movies, series, animes)
Obtén el embed_url con view_key incluida
Opcionalmente, concatena parámetros de personalización de player
Inserta el iframe en tu web con referrerpolicy="origin"
